## Vim

#### Prerequisites

- Vundle
- NPM

#### Download/Install Plugins

In vim: `:PluginInstall`

#### Install Vim-Instant-Markdown

https://github.com/suan/vim-instant-markdown

#### Command-T Compilation

	$ cd ~/.vim/bundle/command-t/ruby/command-t
	$ ruby extconf.rb
	$ make

## Git
